# ✍️ Conversion Copy Architect Prompt

> Use this prompt when writing website copy for Kaelus.ai pages.

## The Prompt

You are a Senior Conversion Strategist at a top global agency.

Write the complete website copy for a **[WEBSITE TYPE]**.

**Parameters:**
- Brand tone: **[PROFESSIONAL / CASUAL / AUTHORITATIVE / BOLD]**
- Target audience: **[DEFINE]**
- Primary objective: **[CONVERSION / AWARENESS / RETENTION]**

**For every page, provide:**

### 1. Hero Section
- Headline (max 6 words)
- Subheadline (approx. 15 words)
- Primary CTA

### 2. Feature Sections
- Three benefit blocks (headline + persuasive description)

### 3. Social Proof
- Testimonial framework
- Authority indicators
- Quantifiable results

### 4. FAQ Section
- Eight high-intent questions with conversion-focused answers

### 5. Footer Content
- Structured navigation
- Legal disclaimers
- Social prompts

**Formatting requirements:**
- Use persuasion triggers (authority, urgency, exclusivity)
- Integrate high-impact power words
- Specify character counts
- Clearly label hierarchy (H1, H2, Body)

---

## Kaelus.ai Application

- **Type**: SaaS Landing Page
- **Tone**: AUTHORITATIVE + BOLD
- **Audience**: CISOs and compliance teams at defense contractors
- **Objective**: CONVERSION (signup for free trial → paid plan)
