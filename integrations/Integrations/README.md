# Hound Shield — Integrations Integrations

Route all AI calls through the Hound Shield proxy to enforce SOC 2, HIPAA, and CMMC Level 2 simultaneously across your Integrations stack.

## Quick Setup

```bash
HOUNDSHIELD_PROXY_URL=https://your-tenant.houndshield.com/proxy
```

Replace your AI provider base URL with the Hound Shield proxy endpoint — no other code changes required.
