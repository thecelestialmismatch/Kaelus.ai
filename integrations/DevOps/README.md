# Kaelus — DevOps Integrations

Route all AI calls through the Kaelus proxy to enforce SOC 2, HIPAA, and CMMC Level 2 simultaneously across your DevOps stack.

## Quick Setup

```bash
KAELUS_PROXY_URL=https://your-tenant.kaelus.online/proxy
```

Replace your AI provider base URL with the Kaelus proxy endpoint — no other code changes required.
