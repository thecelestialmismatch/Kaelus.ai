# Mobile App Design Inspiration

Screenshots captured from Yupp.ai (yupp.ai) for mobile app UX reference.

## Key Design Patterns to Adopt

### Cards Carousel (Model Selection)
- Horizontal swipeable cards per AI model
- Model name, logo, capability tags
- "Try Model →" CTA on each card

### "Every AI for Everyone" Grid
- Icon grid showing supported models (Claude, GPT, Gemini, Mistral, etc.)
- Tagline: "800+ models. Zero compromise. Full compliance."

### Leaderboard UI
- Tab bar: Text · Image · Coding · Live · Security
- Ranking rows with model name, score bar, compliance badge
- Kaelus twist: compliance score per model (not just performance)

### Sign-up Flow
- Google OAuth primary CTA (orange button)
- Minimal friction — no credit card on free tier
- Dashboard preview in hero

## Kaelus Differentiators vs Yupp
- Every model is SCANNED before it sees your data
- Compliance badge on every model card (SOC2 ✓ HIPAA ✓ CMMC ✓)
- Audit trail for every prompt, every model
- Real-time SPRS score visible in mobile dashboard
- BLOCKED prompt notifications via push notification
