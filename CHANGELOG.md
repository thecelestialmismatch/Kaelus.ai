# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-03-10

### Added
- Complete rewrite of the Kaelus.ai compliance firewall agent using Next.js 14 App Router.
- Context-Aware ReAct Engine logic framework mapped.
- Zero-trust visual dashboard and mock analytics.

### Changed
- Migrated codebase to React 18 Server Components for sub-50ms render latency.
- Overhauled root repository structure to adhere to open-source community standards.
- Updated security and contribution documentation.

### Removed
- Legacy project files, temporary drafts, and outdated PRDs.
